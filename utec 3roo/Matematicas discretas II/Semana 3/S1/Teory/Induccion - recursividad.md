#### **Succesiones:**
x1,x2,x3,x4...
Donde cada termino x, es un numero real
x1 = 1 => indice
Puede empezar con cualquier numero entero
#### Succesion recurrente:
Una sucesion que cumple una condicion del tipo:
Xn = f(X1,X2,X3...Xn-1)
Para todo n >= k = k => 0,1,2,3...
